## Escenario
Se desarrolló el escenario 13, que consiste en un sistema de inscripción a cursos.

El programa permite registrar estudiantes, visualizar los cursos disponibles, realizar inscripciones, controlar los cupos, generar una lista de espera cuando un curso se encuentra completo y mostrar estadísticas generales del sistema.

## Funciones del sistema
- Registrar estudiantes.
- Mostrar estudiantes registrados.
- Mostrar cursos disponibles.
- Inscribir estudiantes.
- Controlar cupos.
- Mostrar lista de espera.
- Mostrar estadísticas.
